Fluent Bit

Fluent Bit

Fluent Bit is a lightweight and optimized events data collector for Linux, Embedded Linux, OSX and BSD family operating systems. It have been made with a strong focus on IoT to allow the collection of events from different sources like network protocols, hardware radios and IoT frameworks.

Fluent Bit is part of the Fluentd project ecosystem, it's licensed under the terms of the Apache License v2.0.

This project is made and sponsored by Treasure Data.
